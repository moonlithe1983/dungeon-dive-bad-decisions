# Art Production Plan

Version date: April 14, 2026

This plan covers visual production for the rebooted game.

## Art strategy

Use a two-stage pipeline:

1. AI-assisted exploration for speed
2. professional finalization for consistency, readability, and commercial quality

## What AI can do well

Use Midjourney, ChatGPT image generation, or similar tools for:

- moodboards
- composition studies
- environment concept exploration
- UI tone references
- enemy and class visual ideation
- marketing concept drafts
- rapid style variation testing

This is especially useful while the core loop is still changing.

## What AI should not be trusted to finalize alone

Do not rely on AI alone for:

- final gameplay-critical silhouettes
- enemy readability systems
- final iconography
- consistent character sheets
- animation-ready sprite or frame sets
- final HUD assets
- final store art
- final production asset packs that must match across many surfaces

The rebooted game needs clear visual language, repeatable asset logic, and deliberate cohesion. AI can suggest, but it rarely maintains production consistency without strong human art direction.

## Recommended production path

### Phase 1: visual exploration

Use AI to produce:

- 2 to 3 visual directions
- one moodboard per direction
- sample hero, companion, enemy, and environment treatments
- UI surface reference boards

Decision output:

- pick one art direction for the vertical slice

### Phase 2: vertical-slice production art

For the slice, create only what is necessary:

- one environment kit
- one hero presentation style
- two class presentation variants
- two companion presentation variants
- three to four enemy silhouettes
- one boss visual set
- readable pickups and reward icons
- core HUD and feedback effects

### Phase 3: professional polish

Bring in a professional artist when:

- the slice is mechanically stable enough to stop changing daily
- the visual direction is chosen
- the game needs a production-consistent asset set for gameplay and marketing

Preferred artist responsibilities:

- final style guide
- silhouette language
- color and value hierarchy
- character cleanup
- environment consistency
- iconography system
- final promo and store assets

## Non-negotiables for art

- gameplay readability beats decorative detail
- hazards must read faster than lore text
- rewards must be distinguishable at a glance
- enemy classes and elite states must be visually legible
- UI art must support touch readability on phones
- the visual style must reinforce the corporate-horror identity without making the screen muddy

## Minimum slice art list

- title / splash treatment
- one environment backdrop set
- one room hazard language set
- one player base kit
- two class variants
- two companion cards
- four enemies
- one boss
- pickups for health, currency, and one upgrade type
- core HUD icon set

## Recommendation

Prototype with AI. Ship-quality finalization should involve a professional artist unless the product intentionally accepts a rougher prototype look.
